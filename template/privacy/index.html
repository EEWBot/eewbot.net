プライバシーポリシー

最終更新日: 2026-06-27

EEWBot 開発チーム（以下「当方」といいます）は、Discord Bot「EEWBot」（以下「本Bot」といいます）の公式インスタンス(Discord User ID: 329257498668302346)を運営するにあたり、本ポリシーに従って情報を取り扱います。本Botをサーバーに追加し、または継続して利用する場合、本ポリシーの内容をご確認ください。

1. 適用範囲

本ポリシーは、当方が運営する EEWBot 公式インスタンスに対して適用されます。第三者がセルフホストしたインスタンスについては、当該運営者のポリシーが適用されるため、本ポリシーは適用されません。

2. 取得する情報

本Botは、サービス提供に必要な範囲で、以下の情報を取得する場合があります。

  • Discord上の識別情報（ユーザーID、サーバーID、チャンネルID、スレッドID等）や、付随する権限情報
  • 通知配信に必要な接続情報（Webhook URL、Webhook ID、Webhook Token等）
  • Discord上で本Botに設定されたサーバー内ニックネーム
  • 本Botに設定された通知設定、配信設定、言語設定、フィルター設定等
  • 本Botに対して実行されたコマンドの内容
  • 本Botの動作に関するログ、エラーログ
  • Webサイト、画像、埋め込みコンテンツ等へのアクセスに関する情報（IPアドレス、アクセス日時、リクエスト情報、User-Agent、リファラ等）
  • その他、本Botの機能提供に必要なDiscord上の情報

本Botは、通常のメッセージ内容を常時監視・保存することを目的としていません。ただし、コマンド、ボタン、メニュー、モーダル等を通じてユーザーが入力した内容については、機能提供のために取得・処理する場合があります。

3. 利用目的

取得した情報は、以下の目的で利用します。

  • 本Botの機能提供
  • ユーザー、サーバー、チャンネルごとの通知設定の保存および適用
  • 通知、アラート、各種情報配信の実行
  • 不具合調査、障害対応、ログ確認
  • 不正利用、規約違反、迷惑行為への対応
  • 本Botの品質改善および安定運用
  • ユーザーからの問い合わせ対応

4. 情報の保存

本Botは、機能提供に必要な情報をデータベース、設定ファイル、キャッシュ、ログ等に保存する場合があります。

保存される情報には、ユーザーID、サーバーID、チャンネルID、Webhook情報、通知設定、その他Botの動作に必要な設定情報が含まれる場合があります。

ログについては、障害調査やセキュリティ対応、その他機能改善のために一定期間保存する場合があります。

5. Webhook情報の取り扱い

本Botは、通知配信のためにDiscord Webhookを利用する場合があります。

Webhook URLおよびWebhook Tokenは、第三者に知られると当該Webhookを通じた投稿が可能となる機密情報です。当方は、これらの情報を本Botの機能提供に必要な範囲でのみ利用し、不要になった場合は合理的な範囲で速やかに削除します。

6. 第三者提供

当方は、以下の場合を除き、取得した情報を第三者に提供しません。

  • ユーザーまたはサーバー管理者の同意がある場合
  • 法令に基づく場合
  • 不正利用、セキュリティ上の問題、権利侵害等への対応が必要な場合
  • 本Botの運用に必要な範囲で、外部サービス事業者に情報の処理を委託する場合

7. 外部サービスの利用

本Botの運用にあたり、以下のような外部サービスを利用する場合があります。

  • Discord API
  • CDN、キャッシュ、セキュリティサービス
  • ホスティングサービス
  • データベースサービス
  • ログ管理サービス
  • 監視サービス
  • バックアップサービス

これらの外部サービスでは、サービス提供、コンテンツ配信、キャッシュ、セキュリティ対策、不正アクセス防止、障害調査等に必要な範囲で、IPアドレス、アクセス日時、リクエスト情報、User-Agent、リファラ、Cookieその他の通信に関する情報が取得、保存、処理される場合があります。

また、これらの外部サービスは、日本国外に所在する事業者または日本国外のサーバーを利用して提供される場合があり、上記の情報が日本国外で取得、保存、処理される場合があります。

8. 情報の削除

サーバー管理者または正当な権限を持つユーザーは、本Botに保存された当該サーバーに関する設定情報の削除を依頼できます。

削除依頼を受けた場合、当方は必要に応じて本人確認または権限確認を行った上で、合理的な範囲で対応します。

ただし、以下の情報については、即時削除できない場合があります。

  • バックアップに含まれる情報
  • 障害調査、セキュリティ対応、法令対応のために必要なログ
  • 技術上、即時削除が困難な情報

9. データの保持期間

本Botは、利用目的の達成に必要な期間、取得した情報を保持します。

本Botがサーバーから削除された場合、当該サーバーに関する設定情報は、一定期間経過後または削除依頼に基づき削除される場合があります。

ログの保持期間は、運用上必要な範囲で定め、不要となったものから順次削除または匿名化します。

10. セキュリティ

本Botは、取得した情報について、不正アクセス、漏えい、改ざん、紛失等を防止するため、合理的な安全管理措置を講じます。

ただし、インターネット上の通信、外部サービス、Discord API、ホスティング環境等の性質上、完全な機密性・安全性・可用性を保証するものではありません。

11. 本ポリシーの変更

本ポリシーの内容は予告なく更新されることがあります。

重要な変更を行う場合は、合理的な方法により通知または公表します。

変更後のプライバシーポリシーは掲載された時点で効力が発生するものとします。
